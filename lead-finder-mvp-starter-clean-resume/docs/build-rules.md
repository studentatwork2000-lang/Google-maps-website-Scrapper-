# Build Rules

## Must do

- Use original code only.
- Use Google Places API or mock data.
- Keep the API key server-side.
- Retrieve website fields without opening each Maps card manually.
- Keep CSV export clean.
- Keep UI simple and practical.

## Must not do

- Do not copy code from any paid extension.
- Do not call private scraper backend endpoints.
- Do not bypass any membership/paywall system.
- Do not build a click-each-card scraper.
- Do not rely on Google Maps frontend CSS selectors.
- Do not expose the Google API key in React/Vite frontend env.

## Design principle

Same business outcome, independent implementation.
